fibonacci
===============

Two implementations of the same fibonacci sequence generator, one using BDD and another using the classic development workflow.

## Requirements
'pip install -r requirements.txt'

## How to Run

**BDD**

`cd fibonacci_bdd`

`behave`

**Classic**

`cd fibonacci_old`

`python tests/test_fibonacci.py`
